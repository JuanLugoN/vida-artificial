# Gramaticas (Sistemas de Lindenmayer)

Implementación de los siguientes sistemas de Lindenmayer:
* Binary Tree
* Koch Curve
* Sierpinski Triangle
* Dragon curve
* Fractal plant

### Requisitos
```
$ pip3 install Pillow
$ sudo apt install python3-tk (Si ocupa alguna distribución basada en devian)
```

### Ejecución
```
$ python3 src/l-systems.py
```
